# Sensor-App
This app reads and stores four sensor data such as light, proximity, accelerometer and gyroscope sensors data in SQLite database.  This app gets all sensor data when user clicks on **UPDATE SENSORS** button and then stores the data in **SQLite** database and will update the data to corresponding card. </br>
When user goes to specific sensor card he can see all sensor data lists with corresponding time.
